Webkomponenten Allgemein

Komponenten ermöglichen die Aufteilung der UI in eigenständige, wiederverwendaber Code-Snippets.

Komponenten in React

In React können Webkomponenten durch zwei Arten angelegt werden: Javascript-Klassen und Javascript-Funktionen. Bei der Abbildung als Klasse muss von der Basisklasse React.Component abgeleitet und die render()-Methode implementiert werden:

Bei der Implementierung als Funktion wird direkt JSX mittels return-Statement zurückgegeben. JSX ist eine Mischung aus HTLM und Javascript Code, welches wie HTML aussieht. Mehr Infos dazu gibt es hier

Geschachtelte Komponenten

Es darf nur eine Komponente zurückgegeben werden - in dieser können aber beliebig viele andere Komponenten vorkommen. Sollten auf oberster Ebene einer Komponente mehrere HTML-Tags angeführt werden, müssen diese in einem Element verschlachtelt werden.
